# integer constant formatting in multiple numeric bases


This example demonstrates the use of **integer constant formatting in multiple numeric bases
** in Go.

## Concepts Covered

- integer constants declaration
- formatted output using fmt.Printf
- decimal, binary, and hexadecimal representation of integers
- use of format verbs %d, %b, and %#x for numeric bases


## Purpose

To display predefined integer constants in decimal, binary, and hexadecimal formats using formatted printing for clear comparison of numeric representations.


## Notes

- %#x includes the 0x prefix for hexadecimal output, enhancing readability.
- %b verb outputs the binary representation without leading zeros.
- Using %v for the initial value prints the decimal form by default.
- Constants ensure immutability and clarity in the values being formatted.
