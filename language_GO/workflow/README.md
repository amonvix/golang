# Workflow & Program Execution

This section focuses on **how Go programs execute and make decisions**.

Topics include:

- Introduction of chapter
- Conditional
- Logical Operators
- Modules
- Switch Statement and Comma Ok
- Select for concurrency
- Loops
- Nested Loops
- For Range

The goal is to understand not just _what_ Go does, but _how and why_ it executes code the way it does.
